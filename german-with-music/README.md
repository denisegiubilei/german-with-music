# German with music

Next.js (App Router) app for learning German through music.

## Deployed environment

- Production: [https://german-with-music-4908a3a1e50f.herokuapp.com/](https://german-with-music-4908a3a1e50f.herokuapp.com/)

## Prerequisites

- [Node.js](https://nodejs.org/) 20+ (LTS recommended)
- npm (ships with Node)

## Run locally

From this directory (where `package.json` lives):

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

The dev server hot-reloads when you edit files (for example [`src/app/page.tsx`](src/app/page.tsx) or components under [`src/`](src/)).
